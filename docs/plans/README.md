## 경쟁앱 대비 기능격차 (로드맵, 착수 계획 없음 — 계획16 D절)

코드 조사가 아니라 도메인 지식+웹검색 기반 분석. 별도 계획 문서 없이 여기에만 기록:

- **정기 자동매수("주식 모으기"류)** — 토스증권 대표 기능. Growlio는 DCA 분석만 있고 AUTO는 전부 드리프트 트리거. 실자금 이동 신규 기능이라 신중한 별도 논의 필요([[project_feature_audit_2026-07-06]]과 동일 이유).
